# Support

TriadLLM support is currently lightweight and documentation-first.

## Start Here

For most questions, check:

- [`README.md`](./README.md)
- [`docs/INSTALLATION.md`](./docs/INSTALLATION.md)
- [`docs/CONFIGURATION.md`](./docs/CONFIGURATION.md)
- [`docs/PROVIDERS.md`](./docs/PROVIDERS.md)
- [`docs/TROUBLESHOOTING.md`](./docs/TROUBLESHOOTING.md)
- [`docs/FAQ.md`](./docs/FAQ.md)

## Use GitHub Issues For

- reproducible bugs
- documentation problems
- feature requests
- provider compatibility issues

## Include Useful Context

When asking for help, include:

- your OS
- Python version
- how you installed and launched the app
- relevant profile configuration with secrets removed
- relevant log excerpts with secrets removed
- what you expected
- what actually happened

## Security Issues

Do not report security issues in public issues.

See [`SECURITY.md`](./SECURITY.md).

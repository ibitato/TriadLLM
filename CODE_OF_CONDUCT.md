# Code of Conduct

## Summary

Participation in the TriadLLM project is expected to stay:

- respectful
- direct
- technically constructive
- free of harassment

This applies to issues, pull requests, release discussions, and other project spaces.

## Expected Behavior

- discuss ideas and implementation choices in good faith
- keep criticism focused on code, design, or process
- be clear, specific, and professional
- help keep the repo usable for new contributors

## Unacceptable Behavior

- harassment or intimidation
- personal attacks
- discriminatory or hateful language
- doxxing or publication of private information
- spam, trolling, or deliberately disruptive behavior

## Maintainer Responsibility

Project maintainers may edit, remove, or close content that violates these expectations.

## Reporting

If you need to report conduct issues, contact:

- `ibitato@gmail.com`

Include links, screenshots, or other context if available.
